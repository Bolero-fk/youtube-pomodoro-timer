# YouTube Pomodoro Timer

好きな YouTube 動画を BGM として使える、1 ファイル構成のポモドーロタイマーです。

作業用動画と休憩用動画の URL、作業時間、休憩時間を指定すると、作業時間と休憩時間を自動で切り替えながら再生します。

## Features

- YouTube URL を貼り付けて動画を指定
- 作業時間と休憩時間を分単位で指定
- 作業動画は、作業時間終了時の再生位置を保存
- 次の作業では、保存した位置から作業動画を再開
- 休憩動画は、毎回先頭から再生
- 動画が短い場合は自動でループ再生
- Start / Pause / Resume / Skip / Reset に対応
- 設定をブラウザの localStorage に保存
- GitHub Pages でそのまま公開可能

## Usage

1. ページを開きます。
2. 作業用 YouTube URL を入力します。
3. 休憩用 YouTube URL を入力します。
4. 作業時間と休憩時間を入力します。
5. `Start` を押すとタイマーが開始されます。

## Playback Behavior

### Work Video

作業動画は、作業時間が終わると現在の再生位置を保存します。
次の作業時間では、保存した位置から再開します。
動画が最後まで再生された場合は、自動で先頭からループします。

### Break Video

休憩動画は、休憩時間が終わっても再生位置を保存しません。
次の休憩時間では、常に先頭から再生します。
動画が最後まで再生された場合は、自動で先頭からループします。

## Deploy to GitHub Pages

GitHub Pages で公開する場合は、リポジトリ直下に `index.html` を配置してください。

## Notes

- YouTube の埋め込み再生が許可されていない動画は再生できない場合があります。
- 広告が表示される場合、タイマーや再生体験が想定どおりにならない可能性があります。
- iPad やスマートフォンでは、画面を表示したまま使う方が安定します。
- ブラウザの仕様により、初回再生にはユーザー操作が必要になる場合があります。
